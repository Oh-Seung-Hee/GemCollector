# 유니티 개발 심화주차 프로젝트 

## 개요
- 버전 : 2022.3.2.f1

## 역할분담
- 오승희 - 인벤토리, 아이템
- 조민상 - 플레이어, 몬스터
- 정지엽 - 절차적 맵 생성
- 이호열 - UI, 사운드

## 키 설정
- 이동 : 방향키
- 공격 : C
- 인벤토리 : I
- 설정 : ESC
- 플레이어 스텟창 : S
- 아이템 사용 : 마우스 좌클릭
